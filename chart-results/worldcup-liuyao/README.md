# 世界杯六爻排盘结果归档规范

本目录用于存放按根目录 `世界杯-六爻规则.md` 生成的世界杯六爻纳甲预测结果。

## 文件命名

- 单轮多场比赛：`YYYY-MM-DD-matchday-<编号范围>.md`
- 单场比赛：`YYYY-MM-DD-<A队>-vs-<B队>.md`
- 日期优先使用比赛所在地当地日期；若一份档案包含多个时区，以首场比赛当地日期为准。

## 必填内容

每份档案必须包含：

1. 基本信息：生成日期、规则文件、资料来源、A/B 队对应关系。
2. 时间校正：当地民用时间、UTC 偏移、场地经度、均时差、比赛地真太阳时。
3. 时间数过程：年支序数、农历月数、农历日数、时支序数、上卦、下卦、动爻。
4. 排盘参数：农历日期、四柱、本卦、动爻、变卦、旬空、六神。
5. 完整六爻排盘：爻位、六神、六亲、纳甲、动静、世应、变爻。
6. 世应判断：世应旺衰、生克、冲合、空破、入墓、动变。
7. 得分信号：A/B 独立有效得分信号、无效或被化解信号。
8. 唯一 90 分钟比分：按规则映射产生，不得在归档时人工改分。
9. 边界提示：真太阳时接近时辰边界、节气边界、首哨延迟导致跨时辰等情况。
10. 免责声明：仅供传统文化研究和自我观察参考，不构成投注、投资或确定性决策依据。

## 归档要求

- 每次按世界杯六爻规则输出最终预测前，必须先在本目录写入或更新对应档案。
- 如果同一场比赛因实际首哨时间、场地或规则修正而重排，应新增修订小节，保留原判断和修订原因。
- 不得只保存最终比分；必须保存足以复核结论的时间数过程、六爻排盘和得分信号。
- 若用户提供截图、网页或外部赛程表，应在档案中记录来源描述或 URL。
