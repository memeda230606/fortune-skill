# 世界杯奇门排盘结果归档规范

本目录用于存放按根目录 `世界杯-奇门遁甲规则.md` 生成的世界杯奇门遁甲预测结果。

## 文件命名

- 单轮多场比赛：`YYYY-MM-DD-matchday-<编号范围>.md`
- 单场比赛：`YYYY-MM-DD-<A队>-vs-<B队>.md`
- 日期优先使用比赛所在地当地日期；若一份档案包含多个时区，以首场比赛当地日期为准。

## 必填内容

每份档案必须包含：

1. 基本信息：生成日期、规则文件、资料来源、A/B 队对应关系。
2. 时间校正：当地民用时间、UTC 偏移、场地经度、均时差、比赛地真太阳时。
3. 排局参数：四柱、节气、符头或旬首、上中下元、阴阳遁、局数、空亡、值符、值使。
4. 完整九宫排盘：宫位、门、星、神、天盘干、地盘干、空亡、入墓、击刑、门迫。
5. A 日干宫与 B 时干宫：双方宫势、主要加力和减力因素。
6. 得分信号：A/B 独立有效得分信号、被排除的重复或失效信号。
7. 唯一 90 分钟比分：按规则映射产生，不得在归档时人工改分。
8. 边界提示：真太阳时接近时辰边界、节气边界、首哨延迟导致跨时辰等情况。
9. 免责声明：仅供传统文化研究和自我观察参考，不构成投注、投资或确定性决策依据。

## 归档要求

- 每次按世界杯奇门规则输出最终预测前，必须先在本目录写入或更新对应档案。
- 如果同一场比赛因实际首哨时间、场地或规则修正而重排，应新增修订小节，保留原判断和修订原因。
- 不得只保存最终比分；必须保存足以复核结论的排盘依据和得分信号。
- 若用户提供截图、网页或外部赛程表，应在档案中记录来源描述或 URL。
