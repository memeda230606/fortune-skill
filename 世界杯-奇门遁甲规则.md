【起局前复盘检查】
每次世界杯奇门预测前，必须先完成复盘检查，不得只凭当前盘面临场发挥。

检查顺序：
1. 先读取本主规则文件，确认当前有效规则。
2. 再按当前盘面特征检索 `chart-results/worldcup-qimen/` 下已归档复盘，不要求逐字重读全部档案，但必须检索相似标签和相似结构。
3. 检索关键词至少覆盖当前盘面的核心风险：同宫、中五、值符、值使、开门、景门、伤门、生门、休门、空亡、入墓、击刑、门迫、玄武、天蓬、太阴、三奇得使、直接克制、迟发、补时、开放度、归零、上限封顶。
4. 若当前盘面触发某类已升级主规则，必须按主规则执行。
5. 若只命中单场“观察样本”而未升级主规则，只能作为高风险点或上下限提示；不得直接改写唯一比分，除非当前盘面同时满足该观察样本写明的全部触发条件，并在输出中说明。
6. 若多个复盘样本互相冲突，优先级为：主规则 > 已升级主规则校准 > 多场重复复盘偏差 > 单场观察样本 > 临场直觉。
7. 禁止为了贴合已知真实赛果回读同一场赛后结果。若用户要求赛前预测且禁止联网，不能打开同一场已含真实赛果的档案来反推；只能参考其他历史样本和主规则。
8. 复盘检查不得引入球队强弱、排名、赔率、名气、近期战绩等禁用信息。

复盘检查输出要求：
1. 每场预测必须写“起局前复盘参照”小节。
2. 小节内列出命中的复盘标签或样本类型，例如 `开放度低估`、`上限机械封顶`、`被克方误归零`、`击刑未折损`、`补时迟发样本`。
3. 写明采用的校准、未采用的观察样本和原因。
4. 若没有可比复盘，也必须写“未命中相似复盘，按主规则直断”。

【固定排局规则】
一、时间基准
1. 优先采用实际首哨时刻；没有则采用官方开球时刻。
2. 主盘使用比赛地真太阳时。
3. 若未填写真太阳时，按：
   真太阳时＝当地民用时间＋4×（场地经度－UTC偏移×15°）分钟＋当日均时差。
4. 经度东正西负，UTC偏移包含夏令时。
5. 若真太阳时跨越日期或时辰，按校正后的时间排局。
6. 若比赛发生在节气交接当日，必须依据准确交节时刻确定前一节气或后一节气，不得只按日期判断。

二、排局体系
1. 固定使用时家转盘奇门。
2. 固定使用拆补法定局。
3. 根据比赛真太阳时排出年柱、月柱、日柱、时柱。
4. 从冬至至芒种采用阳遁，从夏至至大雪采用阴遁；节气交界按准确交节时刻处理。
5. 向前寻找最近的甲日或己日作为符头：
   符头地支为子、午、卯、酉者属上元；
   寅、申、巳、亥者属中元；
   辰、戌、丑、未者属下元。
6. 按该节气的上、中、下元局数表确定具体阴遁或阳遁局数。
7. 完整排出地盘三奇六仪、天盘干、九星、八门、八神、值符、值使。
8. 标注旬首、六仪击刑、入墓、门迫、空亡、马星、伏吟、反吟及五不遇时。
9. 中五宫统一寄坤二宫，不得临时改用其他寄宫方式。
   赛后中五校准：若日干或时干落中五，主宫仍按中五寄坤二处理；但若值符、天禽或同一旬首信息携该干落到另一宫，且该宫同时具备开门、景门或伤门、值符、旺相星门、门宫和义或明确吉格，应作为该方辅助主轴纳入计分。空亡、刑格、时格等凶象先作折损，不得在强主轴齐备时直接归零。

【主客用神固定规则】
1. 列前方A固定取日干落宫。
2. 列后方B固定取时干落宫。
3. 日干或时干为甲时，以其旬首所遁六仪代替。
4. 不得因为队伍实力、中立场或赛程称谓改变日干、时干的对应关系。
5. 若日干与时干同宫：
   先区分天盘干和地盘干的状态；
   再以值符宫辅助判断A，以值使宫辅助判断B；
   只能作为同宫细分，不得交换A、B。
   赛后同宫校准：若值使门仍回到同宫，且同宫同时具备开门、景门或伤门之一、丙丁奇、明确吉格或三诈五假等多重进攻符号，即使同宫有空亡、门迫、受刑，也不得直接把双方进球信号归零；应先按值符宫给A、值使宫给B拆分，再把空亡、门迫、受刑作为折损项。若A的值符辅助宫不空不迫，且有生门、值符、丙丁奇或欢怡等稳定转化象，可计低限1球；若B的值使同宫具备开门+丙丁奇+明确吉格且星门不弱，可在折损后保留2至3球候选。
6. 景门用于观察比赛呈现及进球活跃度，庚用于观察对抗阻力，值符和值使用于整体局势复核，但不能替代日干、时干主用神。

【宫势判定】
分别评估A宫和B宫：

加力因素：
1. 宫五行在节令中旺或相。
2. 天盘干得到地盘干、宫、门或星生扶。
3. 临开门、生门、休门，且门宫关系不受迫。
4. 临天冲、天英、天心、天辅等能够形成有效推进或组织的组合。
5. 临九天、值符、六合等有利神，并与本宫组合相符。
6. 临丙、丁奇或吉格，能够直接作用对方宫。
7. 本宫克制对方宫，且自身不空、不墓、不受反克。
8. 对方宫存在明显防守缺口。

减力因素：
1. 宫五行休、囚、死或被对方宫直接克制。
2. 空亡、入墓、击刑、门迫、五不遇时。
3. 临杜门、死门且无旺星、奇仪解救。
4. 天盘干被地盘干反克或构成明显凶格。
5. 伏吟导致行动迟滞。
6. 有利符号重叠于同一来源时只能计算一次。

【统一比分量化】
分别统计A、B的“独立有效得分信号”：

每出现以下一项，可计一个信号：
1. 本方宫旺相，并对对方宫形成有效克制。
2. 本方临开门、景门或伤门，同时得到旺星或丙丁奇配合。
3. 本方临天冲、天英、马星、九天等快速进攻象，且不落空亡、入墓。
4. 对方宫出现空亡、入墓、击刑、门迫或受本方直接克制。
5. 本方形成明确吉格，能够突破对方宫防守。
6. 反吟、冲动或马星造成比赛节奏明显开放，并具体作用于本方进攻。

同一门、星、奇、神构成的同一组合不得拆成多个信号；被空亡、入墓、门迫或严重受克完全化解者不计。

赛后复盘校准：
1. 开门、景门或伤门必须同时具备旺星、丙丁奇、明确吉格、直接克制对方宫或对方空破之一，才可计为进球信号；值符、值使同宫只能增强主导性和机会入口，不能替代旺星或丙丁奇。
2. 对方宫临死门、杜门、天芮或受刑，只能优先解释对方失攻；若对方宫不空、不墓、无门迫，且本方宫不能直接克制对方宫，不得把“对方失攻”自动折算为本方进球。
3. 若本方只有开门、值符、值使等场面优势，但星废、宫不旺、无丙丁奇、无直接克制、对方不空破，应记为威胁信号或场面优势，不计有效得分信号。
4. 伤门克对方宫不必然等于进球。若伤门方本宫星门囚废、地盘或天盘入墓、五不遇时或凶格明显，且缺少旺星、丙丁奇、明确吉格或对方严重空破，则优先判为对抗、犯规、伤停、纪律风险或攻势失控，不计有效得分信号。
5. 天冲、马星等快速进攻象若同宫临休门、玄武，且门状态死、天盘干生助对方日干或对方主用神，应优先判为本方失误、门将/后场处理问题或由快转乱；不得直接计为本方进球，可按条件转化为对方1个低限得分信号。
6. 中五日干方若主宫弱，但值符辅助宫同时具备伤门、值符、携日干、不空、星旺或星相，且辅助宫地盘承接对方时干；同时对方时干主宫虽旺但地盘或天盘入墓，值使宫为死门并见时干入墓象，则对方旺动应判为先强后衰、末段收束或纪律崩盘，本方值符伤门可由低限1球上调为3至4球候选，对方进球上限下修为1球。
7. 时干主宫虽见空亡、入墓、三奇入墓或三奇受刑，若值使门同落该宫，且同宫见太阴、三奇得使或类似阴性贵助组合，不得直接判为0球；应优先判为迟发点球、定位球、争议判罚或非流畅机会，保留1球上限，除非同时被对方主宫强克且无任何吉助。
8. 日干主宫若值符和值使同宫，且天盘干为日干本身或同宫有天辅、欢怡等组织主轴，即使临杜门、空亡、门迫，也不得直接归零；应判为控局但效率受限，保留1球低限。若同时缺少开门、景门、伤门、旺星或丙丁奇，则只保留1球上限，不上调为多球。
9. 日干主宫若同时满足：天盘干为日干本身或主用神直接落宫；临值符或天辅；临开门、景门或伤门之一；同宫见欢怡、丙丁奇、明确吉格或稳定组织主轴之一；并且本宫直接克制对方时干宫，且自身不空不墓，则门迫、星门休囚只能作为转化折损，不得机械封顶2球。若全局另见开门、生门、九天、天冲、太阴惊门或反吟冲动等开放节奏，可列3至4球候选。
10. 时干主宫即使被日干主宫克制，若同时具备时干本宫、奇仪相和、休门或门宫和义，且不是完全空亡、完全入墓或被刑迫尽毁；并且全局有开门、生门、天冲、九天、太阴惊门、点球定位球象或反复开放节奏，不得直接归零。应至少保留1球低限；若同局还出现对方门迫、空亡开放或多次追平象，可列2球高风险。

比分映射固定为：
0个有效得分信号＝0球；
1个有效得分信号＝1球；
2至3个有效得分信号＝2球；
4个及以上有效得分信号＝3球。
只有本方极旺、对方严重空破且同时出现多重攻击格局时，才允许判4球或以上。

赛后校准补充：
若本方已满足4个有效得分信号，且四类信号分别来自：
1. 主用神宫的直接攻击门象；
2. 主用神宫内可独立转化的丙、丁奇或明确吉格；
3. 对方宫不能攻击本方，反而生扶本方或明显失攻；
4. 全局出现开门、生门、天冲、天英、九天、马星、太阴惊门或反吟冲动等开放节奏，并能回扣本方攻势；
同时对方有效得分信号为0，则允许将本方列为4球候选，不再机械封顶为3球。该条为赛后样本校准规则，必须在赛前判读时明确写出触发条件；赛后追加不得追溯记为准确比分命中。

最终比分直接由双方有效信号数量产生：
A球数大于B球数判A胜；
两者相等判平局；
A球数小于B球数判B胜。
宫位生克只用于确认信号有效性，不得在得到比分后人为修改结论。

赛前积分、进球、失球和净胜球只用于小组形势、出线压力和比赛动机复核，不得参与定局、换用神或改变比分映射。球队强弱、FIFA排名、Elo、市场身价、纸面阵容级别、历史名气和投注赔率一律不得纳入推算。

【赛前现实校验层】
现实校验只作上下限和可信度修正，不得改变固定排局、A/B用神、日干/时干对应关系，也不得在没有奇门得分信号时凭现实信息强行造分。现实校验不得使用球队强弱判断。

预测阶段允许纳入以下赛前信息：
1. 主场和半主场：东道主、实际主场、同洲观众、旅途距离、气候适应、时差适应。
2. 阵容与纪律：已确认伤停、停赛、门将变化、红黄牌累积、轮换强度；只记录“人员是否齐整/是否缺位”，不得换算为球队强弱。
3. 比赛动机：小组第几轮、是否必须赢、平局是否可接受、净胜球压力、淘汰或出线形势。
4. 赛程体能：休息天数、长途转场、连续高强度比赛、加时或伤病消耗。
5. 比赛环境：草皮、封闭/露天场、温度湿度、海拔、旅行时区差；只作为事件风险或体能风险，不作为强弱评分。

预测阶段禁止纳入以下信息：
1. FIFA排名、Elo、世界排名、洲际排名、历史战绩名气。
2. 市场身价、纸面阵容级别、球星数量、联赛名气。
3. 博彩赔率、媒体预测、公众热度、所谓强队/弱队标签。
4. 近期战绩、近N场进失球、友谊赛走势；除非用户要求另作现实分析，否则不得进入奇门比分推算。

现实校验使用规则：
1. 若奇门盘势与客观现实条件一致，可提高结论可信度，但不得以球队强弱调整比分方向。
2. 若奇门盘给3球以上，而客观条件显示体能、伤停、动机或环境不支撑高转化，应降一档或标记为高风险；不得用“强队/弱队”作为降档理由。
3. 若奇门盘给某队0球，只能因主场、动机、纪律、对方空墓或盘内值符/值使/旺星/开景伤门等信号检查1球低限；不得因球队名气或排名补球。
4. 现实主场和东道主背景只能在同等盘势或已有奇门信号时加半档；不得单独改变胜平负。
5. 若用户禁止联网，只使用用户提供的现实信息；缺失项必须标记为“未纳入”，不得臆测。

【事件类型判定层】
除比分外，每场必须判断主要进球或非进球事件类型。事件类型只辅助解释比分来源，不替代有效得分信号。

常见事件对应：
1. 开门、景门、伤门配旺星、丙丁奇、九天、天英、天冲：开放战进球、快速推进、射门转化。
2. 太阴、死门、值使、三奇得使、入墓后仍有吉助：迟发点球、定位球、争议判罚、非流畅机会。
3. 伤门、白虎、腾蛇、六仪击刑、门迫：犯规、冲突、伤病、红黄牌、身体对抗升级。
4. 玄武、休门、天冲、马星、门死或生助对方：门将/后场失误、由快转乱、反击未成反送机会。
5. 值使死门、时干入墓、对方值符伤门旺起：后段收束、减员、纪律崩盘、末段连续失球。
6. 空亡：机会虚化、VAR/越位/被取消、场面有而转化少；若空亡与开门、生门、九天、天冲、太阴惊门或反吟冲动同局出现，不得一概判为低转化，应同时列为开放节奏、高总进球或反复追平风险。若空亡落对方防守弱点且本方有强主轴，则可作防线漏象。

事件输出要求：
1. 至少给出每队一个主事件判断，如“开放战低限”“迟发点球候选”“失误送势”“红牌风险”“末段崩盘风险”。
2. 若事件判断与比分判断冲突，优先保持比分映射，并把事件列为风险提示。
3. 对红牌、伤病、点球、VAR等事件只给倾向，不写成确定发生。

【比分上下限修正层】
比分必须先按有效得分信号得到原始比分，再进入上下限修正。修正过程必须写明，不得直接给修正后比分。

下限规则：
1. 主用神宫虽弱，但值符、值使、天辅、欢怡、三奇得使、太阴等形成组织主轴，且不被完全空墓化解，可保留1球低限。
2. 对方宫严重空亡、入墓、击刑、门迫，且本方宫不空不墓、有生门或组织承接，可保留本方1球低限。
3. 值使落本方主宫，哪怕临死门、杜门，也须检查是否对应迟发或非流畅1球低限。
4. 被克方若有时干本宫、奇仪相和、休门或门宫和义，且全局有开放节奏或点球定位球象，不得因受克直接归零，应保留1球低限；只有完全空墓、严重刑迫且无任何吉助时才可归零。

上限规则：
1. 本方仅有休门、生门、值符、值使等场面或组织优势，缺少开门、景门、伤门、旺星、丙丁奇、直接克制或对方空破时，封顶1球。
2. 本方有2至3个有效信号，但自身星门囚废、入墓、门迫或吉象同源重复时，封顶2球。
3. 本方满足4个及以上独立信号，且对方出现严重空墓刑迫，可判3球；若同时有对方纪律崩盘象、体能断档、门将/后场失误象或其他客观事件风险，可列4球候选。
4. 本方主宫虽旺，但地盘或天盘入墓，值使死门或时干入墓明显，应判先强后衰，进球上限下修。
5. 对方有效得分信号为0时，仍需检查太阴、值使、死门、三奇得使是否形成迟发1球；若有，则不得机械归零。
6. 日干主宫若同时具备值符或天辅、开景伤之一、欢怡/丙丁奇/明确吉格之一、直接克制对方宫、不空不墓，即使门迫或星门休囚，也不得机械封顶2球；若全局开放节奏明确，应列3至4球候选。

修正输出格式：
1. 原始信号比分：A X-Y B。
2. 下限修正：列出保留或剔除的低限球。
3. 上限修正：列出封顶、上调或降档原因。
4. 最终唯一比分：A X-Y B。
5. 若存在高风险分歧，写“高风险点”，但仍只能给一个最终比分。

【复盘样本标签】
每次联网核验真实赛果后，必须给该场追加复盘标签，用于后续统计规则偏差。标签不得用于追溯宣称原预测命中。

基础标签：
1. `胜负命中`、`胜负未命中`、`比分命中`、`比分未命中`、`单方进球命中`、`单方进球未命中`。
2. `低估值符`、`高估值符`、`低估值使`、`高估值使`。
3. `空亡误归零`、`空亡未折损`、`入墓误归零`、`入墓未折损`。
4. `高估伤门`、`低估伤门`、`高估生门`、`高估休门`。
5. `同宫误判`、`中五寄宫误判`、`现实条件校验不足`。
6. `红牌纪律样本`、`点球定位球样本`、`门将后场失误样本`、`末段崩盘样本`。
7. `大胜上限低估`、`低信号方上限高估`、`平局漏判`、`平局过程漏判`。
8. `开放度低估`、`上限机械封顶`、`被克方误归零`。

复盘记录要求：
1. 每个样本至少标3个标签：命中标签、主要错因标签、事件类型标签。
2. 若新增规则来自该场，必须标记“赛后校准”，并说明不得追溯为准确比分命中。
3. 后续批量总结时，优先统计高频错因；只有多个样本反复出现的偏差，才允许升级为主规则。

【输出格式】
分别输出样本一和样本二：

1. 民用时间、真太阳时及四柱。
2. 所处节气、符头、上中下元、阴阳遁和局数。
3. 起局前复盘参照：命中的历史复盘标签、采用的主规则校准、未采用的观察样本和原因。
4. 完整九宫排盘。
5. A日干宫与B时干宫的门、星、神、天地盘干及旺衰。
6. 双方宫位生克、空亡、入墓、击刑、门迫、伏吟或反吟。
7. A、B各自的独立有效得分信号及被排除的重复信号。
8. 赛前现实校验层：已纳入信息、未纳入信息、是否修正上下限。
9. 事件类型判定层：开放战、点球/定位球、失误、红牌伤病、末段崩盘等风险倾向。
10. 原始信号比分、上下限修正、最终唯一90分钟比分：A X–Y B。
11. 高风险点、边界提示和免责声明。

【自动存档规则】
1. 每次按本规则完成世界杯奇门遁甲预测后，必须自动存档，不得只在对话中输出结论。
2. 存档目录固定为项目根目录下的 `chart-results/worldcup-qimen/`。
3. 存档文件命名：
   - 单轮多场比赛：`YYYY-MM-DD-matchday-<编号范围>.md`
   - 单场比赛：`YYYY-MM-DD-<A队>-vs-<B队>.md`
   日期优先使用比赛所在地当地日期；若一份档案包含多个时区，以首场比赛当地日期为准。
4. 存档必须包含：
   - 生成日期、规则文件、资料来源、A/B 队对应关系；
   - 当地民用时间、UTC 偏移、场地经度、均时差、比赛地真太阳时；
   - 四柱、节气、符头或旬首、上中下元、阴阳遁、局数、空亡、值符、值使；
   - 起局前复盘参照：检索的复盘标签、采用/未采用的样本、原因；
   - 完整九宫排盘；
   - A 日干宫与 B 时干宫的门、星、神、天地盘干及旺衰；
   - 双方宫位生克、空亡、入墓、击刑、门迫、伏吟或反吟；
   - A、B 各自的独立有效得分信号及被排除的重复或失效信号；
   - 赛前现实校验层、事件类型判定层、原始信号比分、上下限修正；
   - 唯一90分钟比分、边界提示和免责声明；
   - 联网核验后的真实赛果、命中对比、复盘标签、赛后校准说明。
5. 若因实际首哨时间、比赛场地、赛程来源或规则修正导致重排，应在原档案中追加修订小节或新增修订档案，并保留原判断和修订原因。
6. 输出给用户时应同时告知存档路径，便于后续复核。
