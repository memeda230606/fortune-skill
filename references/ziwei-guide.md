# 紫微斗数解盘指南

本文件供 LLM 分析紫微斗数命盘时参考。结合 `ziwei-chart.mjs` 输出的 JSON 数据使用。

---

## 一、十二宫含义

紫微斗数以十二宫为框架，每宫代表人生的一个面向。分析时需结合宫内主星、辅星、四化及三方四正综合判断。

| 宫位 | 含义 | 分析要点 |
|------|------|---------|
| 命宫 | 先天性格、才能、外在表现 | 核心宫位，决定人生基调。看主星组合判断格局高低 |
| 兄弟宫 | 兄弟姐妹关系、合作伙伴 | 也看同事关系、平辈互动 |
| 夫妻宫 | 婚姻感情、配偶特质 | 看主星判断配偶类型，四化看感情变化 |
| 子女宫 | 子女缘分、生育、下属 | 也看桃花、性生活、投资（子女为财之库） |
| 财帛宫 | 财运、理财方式、收入模式 | 看赚钱能力和方式，非财富总量 |
| 疾厄宫 | 健康状况、体质、灾厄 | 看先天体质弱点和易患疾病 |
| 迁移宫 | 外出运、社交、外在机遇 | 命宫的对宫，看外在环境和贵人运 |
| 交友宫（奴仆宫） | 朋友、下属、社交圈 | 看人际关系质量和助力 |
| 事业宫（官禄宫） | 事业发展、工作类型、社会地位 | 看适合的职业方向和事业成就 |
| 田宅宫 | 不动产、家庭环境、居住 | 也看家族背景、财库 |
| 福德宫 | 精神生活、兴趣爱好、内心世界 | 看精神层面的满足感和生活品质 |
| 父母宫 | 父母关系、长辈缘、教育 | 也看与上司的关系、文书运 |

### 星曜亮度

星曜在不同宫位有不同亮度，影响其发挥：

| 亮度 | 含义 | 影响 |
|------|------|------|
| 庙 | 最旺，能量最强 | 星曜优点充分发挥 |
| 旺 | 次旺，能量较强 | 优点为主，略有不足 |
| 得地 | 中等，能量平衡 | 优缺点并存 |
| 利 | 偏弱，能量不足 | 缺点开始显现 |
| 平 | 较弱 | 缺点较明显 |
| 不得地 | 弱 | 缺点突出 |
| 落陷 | 最弱，能量最差 | 缺点充分暴露，需辅星补救 |

## 三、四化含义

四化是紫微斗数最核心的动态分析工具。每个天干对应四颗星的四化，代表能量的流动方向。

### 四化基本含义

| 四化 | 含义 | 能量方向 | 关键词 |
|------|------|---------|--------|
| 化禄 | 财禄、顺利、增加 | 给予、获得 | 钱财、享受、人缘、顺利 |
| 化权 | 权力、掌控、竞争 | 争取、主导 | 权力、能力、固执、霸道 |
| 化科 | 名声、贵人、文雅 | 展现、被看见 | 名声、考试、贵人、文书 |
| 化忌 | 执着、困扰、损失 | 纠缠、不顺 | 执着、损失、是非、灾厄 |

### 四化分析要点

1. **生年四化**：先天命格的基本能量分布，影响一生
2. **大限四化**：当前大限（10年）的运势主题
3. **流年四化**：当年的运势重点
4. **自化**：宫位自身产生的四化，代表该宫的内在变化
5. **飞化**：从一宫飞入另一宫的四化，代表宫与宫之间的能量互动

### 化忌特别说明

化忌是最需要关注的四化：
- 化忌入某宫 → 该宫事务容易出问题、不顺
- 化忌入对宫（冲） → 破坏力更大
- 双忌叠加 → 该领域问题严重
- 化忌也代表执着和在意，不全是负面

## 四、三方四正分析方法

三方四正是紫微斗数最重要的分析框架，用于综合评估某一宫位的整体状况。

### 定义

- **三方**：某宫位与其前后各隔三宫的两个宫位，形成三合关系
- **四正**：三方加上对宫（隔六宫），共四个宫位

### 十二宫的三方四正

| 主宫 | 三方 | 对宫（四正） |
|------|------|-------------|
| 命宫 | 财帛宫、事业宫 | 迁移宫 |
| 兄弟宫 | 交友宫、田宅宫 | 疾厄宫 |
| 夫妻宫 | 事业宫、迁移宫 | 福德宫 |（注意：此为夫妻的三方四正）
| 子女宫 | 田宅宫、疾厄宫 | 父母宫 |
| 财帛宫 | 命宫、事业宫 | 福德宫 |
| 疾厄宫 | 兄弟宫、父母宫 | 子女宫 |（注意：此为疾厄的三方四正，非标准三合）

> **注意**：iztro 的 JSON 输出中已包含每个宫位的三方四正信息，可直接使用。

### 分析方法

1. **看主宫**：主星组合决定基本格局
2. **看三方**：三方宫位的星曜会照主宫，增强或削弱主宫能量
3. **看对宫**：对宫星曜直接冲照主宫，影响力仅次于同宫
4. **看四化**：四化飞入三方四正的情况，决定吉凶变化
5. **综合判断**：三方四正整体吉星多则吉，凶星多则凶

### 重要组合

- **命宫三方四正**（命、财、官、迁）：决定人生整体格局
- **财帛三方四正**（财、命、官、福）：决定财运
- **事业三方四正**（官、命、财、夫）：决定事业发展

## 五、大限流年解读要点

### 大限（十年运）

1. **大限宫位**：大限走到哪个宫位，该宫位就成为"大限命宫"
2. **大限四化**：大限天干产生的四化，是该十年的运势主题
3. **叠宫效应**：大限命宫与原局宫位重叠，两者能量叠加
4. **大限三方四正**：以大限命宫为中心，重新审视三方四正

### 流年

1. **流年命宫**：流年地支所在的宫位
2. **流年四化**：流年天干产生的四化
3. **三重叠加**：原局 + 大限 + 流年，三层能量叠加分析
4. **流年斗君**：更精确的流月分析起点

### 分析步骤

1. 确定当前大限宫位和天干
2. 排出大限四化，看飞入哪些宫位
3. 确定流年命宫和天干
4. 排出流年四化
5. 将原局、大限、流年三层四化叠加分析
6. 重点关注化忌的位置和冲照关系

### 吉凶判断原则

- 禄权科入三方四正 → 该领域顺利
- 化忌入三方四正 → 该领域需注意
- 化忌冲命宫或财帛宫 → 该时期较为不顺
- 大限化忌 + 流年化忌同宫 → 双忌叠加，问题集中爆发
- 原局化禄 + 大限化忌同宫 → 禄忌交战，先好后坏或好坏参半

## 六、分析流程建议

使用 `ziwei-chart.mjs` 输出的 JSON 进行分析时，建议按以下顺序：

1. **看命宫主星**：确定命格基调
2. **看命宫三方四正**：评估整体格局
3. **看生年四化分布**：了解先天能量配置
4. **逐宫分析**：按命、财、官、夫、子、田等顺序
5. **看当前大限**：当前十年运势主题
6. **看当前流年**：今年运势重点
7. **综合研判**：结合以上信息给出整体分析

> **提示**：紫微斗数重在"活盘"分析（四化飞星、大限流年叠加），而非死板地看单颗星的吉凶。分析时要注重动态变化和宫位之间的互动关系。
