# 六爻纳甲占问指南

本文件供 LLM 在解释 `scripts/liuyao-chart.py` 输出时参考。脚本负责起卦、装卦和结构化字段，LLM 不手工重排。

## 起卦输入

- 六爻用于具体问题，不替代八字/紫微本命分析。
- 正式六爻优先使用用户手动摇卦得到的六个爻值，自下而上输入：
  - `6`：老阴，动爻，变阳
  - `7`：少阳，静爻
  - `8`：少阴，静爻
  - `9`：老阳，动爻，变阴
- 时间起卦为辅助入口，脚本标记为 `sourceMethod=time_meihua_to_najia`。解释时必须说明该口径，不把它等同于用户亲自摇卦。
- 默认按起卦地点真太阳时；标准时间模式只用于对照。

## 结构化字段

- `hexagram`：本卦，含卦名、卦宫、六亲、纳甲。
- `changedHexagram`：变卦；无动爻时可为空。
- `hiddenSpirit`：伏神；用于补看伏藏的用神或缺失六亲。
- `lines[]`：六爻自下而上，每爻含六亲、六神、纳甲、动静、世应。
- `datePillars`、`xunKong`：月日时干支和旬空，用于判断旺衰、空亡和应期。
- `shiYing`：世爻代表问卦者或自身位置，应爻代表对方、环境或所问对象。

## 判断顺序

1. 明确占问主题，确定用神：
   - 事业/职位/考试：官鬼、父母。
   - 项目/合同/文书：父母、官鬼、世应。
   - 财务/交易/收益：妻财。
   - 感情/合作：世应、财官、六合与冲克。
   - 健康/压力：官鬼、子孙、世爻旺衰；只作文化观察，不作医学判断。
2. 看世应关系：生合、冲克、空亡、动变。
3. 看用神旺衰：月建、日辰、动爻生克、旬空、墓绝。
4. 看原神、忌神、仇神是否发动，以及是否化进、化退、化回头生克。
5. 看动爻与变卦，不只看本卦卦辞。
6. 有伏神时，说明“用神伏藏、事情不显、需条件引出”，不要直接断无。

## 输出边界

- 六爻结论应写成“当前条件下的倾向”，并说明触发条件与现实验证点。
- 不使用“必然发生、一定成功、一定失败”等绝对断语。
- 财务、健康、法律、重大人生决策必须提醒以专业意见和现实信息为准。
- 结尾必须附免责声明：仅供传统文化研究和自我观察参考，不构成专业建议。
