# 八字命理分析指南

本文件供 LLM 分析八字命盘时参考。结合 `bazi-chart.mjs` 和 `bazi-classic.py` 输出的 JSON 数据使用。

---

## 一、日主强弱判断方法

日主（日柱天干）是八字分析的核心。判断日主强弱是确定喜用神的前提。

### 三个维度

#### 1. 得令（月令）

月令是八字中最重要的力量来源，占整体力量的 40% 左右。

| 日主 | 得令（旺相） | 失令（休囚死） |
|------|-------------|---------------|
| 甲乙木 | 寅卯月（春）、亥子月（水生木） | 申酉月（秋金克木）、辰戌丑未月（土耗木） |
| 丙丁火 | 巳午月（夏）、寅卯月（木生火） | 亥子月（冬水克火）、申酉月（金耗火） |
| 戊己土 | 辰戌丑未月（四季）、巳午月（火生土） | 寅卯月（春木克土）、亥子月（水耗土） |
| 庚辛金 | 申酉月（秋）、辰戌丑未月（土生金） | 巳午月（夏火克金）、寅卯月（木耗金） |
| 壬癸水 | 亥子月（冬）、申酉月（金生水） | 辰戌丑未月（土克水）、巳午月（火耗水） |

#### 2. 得地（地支通根）

日主在四柱地支中有根（本气、中气、余气含有与日主同类的五行）。

- **本气通根**：力量最强（如甲木见寅中甲木）
- **中气通根**：力量中等（如甲木见辰中乙木）
- **余气通根**：力量较弱（如甲木见未中乙木）
- **根多根深**：多个地支有根，日主有力
- **无根**：四柱地支无同类五行，日主虚浮

#### 3. 得势（天干透出）

其他天干中有与日主同类（比肩、劫财）或生日主（正印、偏印）的五行。

- **比劫帮身**：其他天干有同类五行
- **印星生身**：其他天干有生日主的五行
- **得势**：比劫印星多，日主有力
- **失势**：官杀财星多，日主受克泄

### 综合判断

| 条件 | 判断 |
|------|------|
| 得令 + 得地 + 得势 | 身极旺（可能从强） |
| 得令 + 得地 或 得令 + 得势 | 身旺 |
| 仅得令 或 仅得地 | 身中和偏旺 |
| 失令 + 失地 + 失势 | 身极弱（可能从弱） |
| 失令但得地得势 | 需细看，可能中和 |

> **提示**：`bazi-classic.py` 输出的 `strength.score` 和 `strength.hasRoot` 可作为参考，但最终判断需结合整体命局。

## 二、十神含义与六亲关系

十神是八字分析的核心概念，以日主为中心，其他天干地支与日主的关系。

### 十神含义

| 十神 | 阴阳 | 与日主关系 | 核心含义 | 代表事物 |
|------|------|-----------|---------|---------|
| 比肩 | 同阴阳 | 同我者 | 自我、竞争、独立 | 兄弟、朋友、同行 |
| 劫财 | 异阴阳 | 同我者 | 争夺、合作、冲动 | 兄弟、合伙人、竞争者 |
| 食神 | 同阴阳 | 我生者 | 才华、享受、表达 | 才艺、口福、子女（女命） |
| 伤官 | 异阴阳 | 我生者 | 叛逆、创新、聪明 | 技术、表演、子女（女命） |
| 偏财 | 同阴阳 | 我克者 | 意外之财、大方、风流 | 父亲、情人、投资收益 |
| 正财 | 异阴阳 | 我克者 | 稳定收入、节俭、务实 | 妻子（男命）、薪资 |
| 七杀（偏官） | 同阴阳 | 克我者 | 压力、魄力、权威 | 小人、意外、丈夫（女命） |
| 正官 | 异阴阳 | 克我者 | 规矩、名声、约束 | 上司、丈夫（女命）、地位 |
| 偏印（枭神） | 同阴阳 | 生我者 | 偏门学问、孤僻、多思 | 继母、偏门技艺 |
| 正印 | 异阴阳 | 生我者 | 学问、保护、慈爱 | 母亲、学历、贵人 |

### 六亲关系

| 六亲 | 男命十神 | 女命十神 |
|------|---------|---------|
| 父亲 | 偏财 | 偏财 |
| 母亲 | 正印 | 正印 |
| 兄弟 | 比肩 | 比肩 |
| 姐妹 | 劫财 | 劫财 |
| 妻子 | 正财 | — |
| 丈夫 | — | 正官（或七杀） |
| 儿子 | 七杀 | 食神 |
| 女儿 | 正官 | 伤官 |

## 三、格局判定方法

格局是八字分析的高级框架，决定命局的层次和发展方向。

### 取格原则（子平真诠）

1. **以月令为主**：月令透干者为格
2. **透干优先**：月令本气透干 > 中气透干 > 余气透干
3. **不透看藏**：月令藏干不透，以本气取格
4. **月令为比劫**：不成格，看其他透干取外格

### 八正格

| 格局 | 条件 | 喜忌 |
|------|------|------|
| 正官格 | 月令透正官 | 喜财生官、印护官；忌伤官破格、七杀混杂 |
| 偏官格（七杀格） | 月令透七杀 | 喜食神制杀、印化杀；忌杀无制化、财生杀旺 |
| 正财格 | 月令透正财 | 喜官星护财；忌比劫争财、劫财破格 |
| 偏财格 | 月令透偏财 | 喜官星护财；忌比劫争财 |
| 正印格 | 月令透正印 | 喜官星生印；忌财星破印 |
| 偏印格 | 月令透偏印 | 喜官杀生印；忌财星破印、食神被夺（枭神夺食） |
| 食神格 | 月令透食神 | 喜财星泄秀；忌偏印夺食 |
| 伤官格 | 月令透伤官 | 喜财星泄秀、印星制伤；忌见官（伤官见官） |

### 特殊格局

| 格局 | 条件 | 特点 |
|------|------|------|
| 从强格 | 日主极旺，全局比劫印星 | 顺其旺势，忌官杀财星 |
| 从弱格 | 日主极弱，全局官杀财星 | 顺其弱势，忌比劫印星 |
| 从财格 | 日主弱，财星极旺 | 以财为用 |
| 从杀格 | 日主弱，七杀极旺 | 以杀为用 |
| 从儿格 | 日主弱，食伤极旺 | 以食伤为用 |
| 化气格 | 日干合化成功 | 以化神为用 |
| 专旺格 | 曲直、炎上、稼穑、从革、润下 | 一行独旺 |

## 四、喜用神确定原则

喜用神是八字分析的核心结论，决定趋吉避凶的方向。

### 基本原则

1. **扶抑法**（最常用）：
   - 身旺 → 喜克泄耗（官杀、食伤、财星）
   - 身弱 → 喜生扶（印星、比劫）

2. **调候法**（穷通宝典）：
   - 寒命（冬生）→ 喜火暖局
   - 热命（夏生）→ 喜水润局
   - 调候用神优先级高于扶抑用神

3. **通关法**：
   - 两行相战（如金木交战）→ 取通关之神（水，金生水生木）

4. **病药法**：
   - 找出命局的"病"（最大问题），"药"即用神

### 确定步骤

1. 判断日主强弱
2. 确定格局
3. 看月令调候需求（参考穷通宝典）
4. 综合扶抑、调候、通关，确定喜用神
5. 喜用神在命局中有力 → 格局高；无力 → 格局低

## 五、大运流年分析要点

### 大运

- 每步大运管十年，天干管前五年，地支管后五年
- 大运与原局发生作用：合、冲、刑、害
- 大运引入新的十神，改变命局力量对比
- 行喜用神运 → 顺利；行忌神运 → 不顺

### 流年

- 流年天干为表象，地支为实质
- 流年与大运组合，再与原局作用
- **岁运并临**：流年干支与大运干支相同，吉凶加倍
- **天克地冲**：流年与大运天克地冲，该年变动大

### 重要流年关系

| 关系 | 含义 | 影响 |
|------|------|------|
| 流年冲日柱 | 冲动日主 | 自身变动大（搬家、换工作、感情变化） |
| 流年冲年柱 | 冲动太岁 | 家庭、长辈方面变动 |
| 流年合日主 | 合住日主 | 有新的关系或合作，也可能被绊住 |
| 流年犯太岁 | 值太岁、冲太岁、刑太岁 | 该年需谨慎 |
| 伏吟 | 流年与某柱相同 | 重复、反复之象 |
| 反吟 | 流年与某柱天克地冲 | 变动、冲突之象 |

### 分析步骤

1. 确定当前大运干支和剩余年数
2. 判断大运十神和对原局的影响
3. 确定当年流年干支
4. 分析流年与大运的组合关系
5. 分析流年与原局四柱的作用关系
6. 综合判断该年运势主题

## 六、常见格局解读

### 身旺用官杀

- 日主旺，以官杀为用神
- 适合体制内、管理岗位
- 有约束力，做事有规矩
- 大运行官杀运时事业有成

### 身旺用财星

- 日主旺，以财星为用神
- 适合经商、投资
- 财运好但需要努力争取
- 大运行财运时收入增加

### 身弱用印星

- 日主弱，以印星为用神
- 适合学术、文化、教育
- 有贵人相助
- 大运行印运时学业事业顺利

### 身弱用比劫

- 日主弱，以比劫为用神
- 适合合伙、团队合作
- 需要朋友帮助
- 大运行比劫运时有助力

### 食伤生财

- 食伤旺，财星有力
- 靠才华技术赚钱
- 适合创意、技术、自由职业
- 食伤为源头，财为结果

### 杀印相生

- 七杀旺，有印星化杀
- 有魄力有学问
- 适合军警、法律、高管
- 格局较高，有权威

### 伤官配印

- 伤官旺，有印星制伤
- 聪明但有约束
- 适合学术研究、专业技术
- 印星是关键，制住伤官才成格

### 财官双美

- 正财正官都有力
- 格局端正，事业财运俱佳
- 适合体制内或大企业
- 男命尤佳（财为妻，官为地位）

## 七、分析注意事项

1. **不要孤立看单柱**：八字是一个整体，四柱之间有生克制化关系
2. **月令最重**：月令是八字的提纲，力量最大
3. **藏干要看**：地支藏干是隐藏的力量，不可忽视
4. **合化要慎重**：天干五合、地支六合是否真正化成，需要条件
5. **刑冲要注意**：地支刑冲会改变力量格局
6. **大运流年是动态**：原局是先天，大运流年是后天变化
7. **调候不可忽略**：冬生喜火、夏生喜水，调候用神很重要
8. **参考经典**：分析时引用经典典籍的论断，增加可信度

> **提示**：`bazi-chart.mjs` 提供精确的排盘数据（四柱、十神、五行、大运），`bazi-classic.py` 提供三命通会论断和格局判定。两者结合使用，数据互补。
